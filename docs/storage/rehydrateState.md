**Notes**: Three items are fetch (`adapter.getItem`) : transactions, wallets and chains data.

| parameters             | type              | required       | Description                                                            |

Returns: Void   
Emit: `REHYDRATE_STATE_SUCCESS/REHYDRATE_STATE_FAILED` event
