**Usage**: `mnemonicToSeed(mnemonic, password)`    
**Description**: Generate the seed from mnemonic

Parameters: 

| parameters        | type          | required       | Description                                      |  
|-------------------|---------------|----------------| -------------------------------------------------|
| **mnemonic**      | Mnemonic      | yes            | A mnemonic                                       |
| **password**      | string        | no             |                                                  |

Returns : {String} hex seed
