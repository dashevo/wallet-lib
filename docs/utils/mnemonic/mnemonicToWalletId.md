**Usage**: `mnemonicToWalletId(mnemonic)`    
**Description**: Generate the deterministic wallet id based on mnemonic

Parameters: 

| parameters        | type          | required       | Description                                      |  
|-------------------|---------------|----------------| -------------------------------------------------|
| **mnemonic**      | Mnemonic      | yes            | A mnemonic                                       |

Returns : {String} hex wallet id
